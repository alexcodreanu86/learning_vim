Traveling Directions:  h - left  j - down  k - up  l - right  w - beggining of next word  W - beggining of next word but only counts white spaces as separators (not . , ! ...)  e - end of current or next word  E - End of word separated only by white spaces  b - beggining of word or previous word  B - Beggining of word or previous word separated only by white spaces  ge - ending of previous word   gE - ending of previous word separated only by white spaces  0 - beggining of the line  $ - end of the line  f <symb> - takes you to the next "symb" character  t <symb> - takes you to the symbol before next "symb" character  F <symb> - takes you to the previous "symb" character  T <symb> - takes you to the character after the previoys "symb" character  ; - repeat previous traveling command  ct <symb> - change everything from cursor to first "symb" character  c2t <symb> - change everything from cursot to second "symb" character (2 can be any noumber ex : "c100t /") Tabs:  link: http://vim.wikia.com/wiki/Using_tab_pages  :tabs         list all tabs including their displayed windows  :tabm 0       move current tab to first  :tabm         move current tab to last  :tabm {i}     move current tab to position i+1  :tabn         go to next tab  :tabp         go to previous tab  :tabfirst     go to first tab  :tablast      go to last tab  :tabedit {file}   edit specified file in a new tab  :tabfind {file}   open a new tab with filename given, searching the 'path' to find it  :tabclose         close current tab  :tabclose {i}     close i-th tab  :tabonly          close all other tabs (show only the current tab)  :tab ball         show each buffer in a tab (up to 'tabpagemax' tabs)  :tab help         open a new help window in its own tab page  :tab drop {file}  open {file} in a new tab, or jump to a window/tab containing the file if there is one  :tab split        copy the current window to a new tab of its own  When starting Vim, the -p option opens each specified file in a separate tab (up to the value of the 'tabpagemax' option). Examples:  vim -p first.txt second.txt  gvim -p *.txt
